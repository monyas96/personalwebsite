Ukrainian Refugees Sentiment Analysis

Real-time social media analytics for humanitarian response in the Netherlands

Overview

Built a real-time social media analytics pipeline to assess public sentiment on Ukrainian refugees in the Netherlands and understand how digital discourse shapes refugee decision-making. The project combined transformer-based NLP with longitudinal analysis to surface patterns relevant to humanitarian response and policy.

What I Built

Twitter Data Pipeline

Real-time data collection using Python and Tweepy, targeting geolocated tweets in Dutch and English.

Transformer Classification

Integrated Hugging Face's cardiffnlp/twitter-roberta-base-sentiment model for sentiment classification.

Longitudinal Tracking

Mapped sentiment spikes to political decisions, media coverage, and border policy changes over time.

Push-Pull Analysis

Explored Ukrainian-language tweets to understand how refugees describe decisions to flee and perceptions of destinations.

Technical Approach

  • Data Collection: Twitter API with geolocation filters and multilingual keyword tracking
  • Classification: CardiffNLP's RoBERTa model fine-tuned on Twitter sentiment
  • Analysis: Time series analysis correlating sentiment with external events
  • Ethics: Careful consideration of bias, anonymity, and representativeness

Policy Impact

The analysis provided insights into how public discourse shapes the refugee experience and policy environment. Findings highlighted the importance of monitoring digital sentiment for humanitarian response planning and the ethical considerations of using social media data for vulnerable populations.
