Nexus — Data Governance & Data Diplomacy

Building an end-to-end data architecture and measurement system for the UN Office of the Special Adviser on Africa (OSAA)

Overview

The Nexus framework represents a strategic initiative by UN OSAA to integrate peace, security, development, and humanitarian dimensions in African policy advisory. I was tasked with translating this high-level vision into a rigorous, data-driven system that could support real-time decision-making for Least Developed Countries (LDCs) and African nations.

What I Built

Logical Framework

Translated high-level strategic concepts into a structured framework mapping themes → topics → indicators, creating a clear hierarchy for measurement.

Data Pipeline

Built a fully automated pipeline using Python and GitHub Actions, feeding into a Lakehouse architecture with curated data marts.

Nexus Dashboard

Designed and implemented dual dashboards (Tableau + Streamlit) for cross-country indicator tracking and interactive exploration.

Measurement Framework

Developed an evidence-informed framework identifying optimal and proxy indicators when primary data was unavailable.

Technical Approach

  • Architecture: Lakehouse pattern with raw → staging → curated data layers
  • Automation: GitHub Actions for scheduled data refresh and quality checks
  • Visualization: Tableau for executive dashboards, Streamlit for analyst exploration
  • Data Sources: UN databases, World Bank indicators, regional statistical offices

Policy Impact

The system now supports real-time advisory and policymaking processes, providing issue breakdowns and targeted recommendations for African development priorities. It bridges the gap between complex data and actionable policy guidance.
