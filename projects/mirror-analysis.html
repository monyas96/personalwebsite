Detecting trade mis-invoicing using UN COMTRADE data and GFI methodology

Overview

Developed a high-performance analytical workflow to detect illicit financial flows (IFFs) via trade mis-invoicing, using UN COMTRADE data and the Global Financial Integrity (GFI) methodology. The project enabled evidence-based policy insights for Africa's domestic resource mobilization agenda.

What I Built

Mirror Analysis

Conducted Partner Country Method (PCM) analysis to estimate trade gaps between African countries and their trading partners.

Automated Pipeline

Built R scripts and DuckDB infrastructure to process >5GB of HS6-level trade data across a 10-year period.

Parquet Optimization

Converted UN COMTRADE API outputs to Parquet format for efficient querying and storage.

Policy Outputs

Computed disaggregated IFF estimates by country, year, and commodity using standardized CIF‚ÄìFOB adjustments.

Technical Approach

  • Data Source: UN COMTRADE bilateral trade flows at HS6 commodity level
  • Methodology: Global Financial Integrity's Partner Country Method
  • Infrastructure: R for analysis, DuckDB for data processing, Parquet for storage
  • Scale: 10-year time series, multiple African countries, >5GB raw data

Policy Impact

The results challenged mainstream narratives on Africa's debt by demonstrating the structural impact of capital flight on debt-to-GDP ratios. Outputs fed into internal advisory notes and briefings for the UN Under-Secretary-General (OSAA), contributing to evidence-based advocacy on Africa's fiscal challenges.
