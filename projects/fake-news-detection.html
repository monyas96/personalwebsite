Disentangling Truth from Noise

An interpretable machine learning approach to fake news detection

Overview

This project designed an interpretable machine learning pipeline to detect fake news, prioritizing transparency and explainability alongside performance. The goal was to build tools that could help protect democratic information ecosystems while remaining understandable to non-technical stakeholders.

What I Built

Data Pipeline

Merged and cleaned fake vs. real news datasets from KaggleHub into a structured corpus for analysis and modeling.

Feature Engineering

Engineered textual (TF-IDF, n-grams), temporal (publication trends), and structural (title-body similarity) features.

ML Pipeline

Built and evaluated multiple classifiers in Scikit-learn: Logistic Regression, Random Forest, and Gradient Boosted Trees.

Explainability

Prioritized model transparency through coefficient visualization, feature importances, and clear classification rationale.

Technical Approach

  • Exploratory Analysis: Title/body length distributions, word frequency analysis, publication trends
  • Feature Set: TF-IDF vectors, n-gram features, punctuation patterns, coherence metrics
  • Models: Logistic Regression (baseline), Random Forest (interpretability), GBT (performance)
  • Evaluation: Precision, recall, F1-score with emphasis on false positive minimization

Policy Impact

The project framed misinformation detection for real-world impact in democratic ecosystems, demonstrating how technical tools can support information integrity while maintaining transparency about their limitations and potential biases.
